# 一、先说 Claude Code 是啥（核心前提）

Claude Code 是一个 AI 编程 Agent：

* 在终端/IDE里运行
* 能**理解整个代码库**
* 可以：

  * 写代码 / 改代码
  * 跑命令
  * 修 bug
  * 自动提交 Git
* 你只要用自然语言说需求，它自己规划并执行

👉 本质：
**不是“代码补全工具”，而是“能自己干活的程序员代理”** ([GitHub][1])

### 1️⃣ 把官方项目“简化成可直接运行”

👉 本质：
**= 官方源码 + 工程化处理 + 可运行环境**
